# 📡 Cable Operator Service Management System

A full-stack web application that helps users report cable issues, assign operators, and track service status in real-time.

---

## 🚀 Live Demo
👉 https://cable-operator-app.onrender.com

---

## 💻 GitHub Repository
👉 https://github.com/Deva517/Cable_Operator_project

---

## 📌 Features

- 🔐 User login with phone number
- 📡 Select network & report issues
- 👨‍🔧 Operator assignment system
- 📊 Job creation & tracking
- 📜 Job history table
- 🔄 Operator workload management
- ⚡ Status updates (Completed / Pending)
- 🌙 Automatic midnight reset system
- 🎉 Smooth UI animations

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: SQLite  
- Deployment: Render  

---

## ⚙️ How It Works

1. User logs in using phone number  
2. Selects problem and network  
3. System assigns an available operator  
4. Job is created and stored in database  
5. Operator status updates dynamically  
6. User marks job as completed  
7. Job history is updated instantly
